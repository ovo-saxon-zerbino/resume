Dissertation: Using Trivial Objects to Track Animal Movement

For my dissertation I worked with the Interactions group at Bristol University on the Gorilla Game Lab project. My work was written in C++ built on the OpenFrameworks platform. My project uses classical computer vision techniques to track animal movement and interaction with the objects introduced by the team to provide data to determine how effective different designs were at getting the animals attention.

Record Keeper for Discogs

The goal of the project is to use Discogs.com public APIs to track desirable records and notify users when records they are interested in become available. The backend architecture is written in Node.js, and run in Docker using MariaDB to store user information. The frontend was developed as a Vue.js app to allow greater interactivity to users and let me explore animation and motion design.
